Mum Facts

As we prepare for the 2014 Mummers Parade, here are some facts regarding the top five string bands of 2013:

  1. No band has ever won going up 2nd.
  2. No band has ever won with a gold miner theme.
  3. No band has won going up 1st since 1933 (sort of).
  4. South Philly, a band that prides itself on its show and usage of large props, has not won since large props were introduced.
  5. Ferko has never gone this long without a win. Their 1999-2013 drought is now equal to Fralinger’s 1988-2002 drought.

BONUS: Avalon has never won.

Check out the String Band Database for more information.

Update: Hey, look! There was a formal study done on the subject: http://pirate.shu.edu/~rotthoku/papers/Order%20Matters.pdf
